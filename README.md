# VoteHub

VoteHub is a secure digital election platform for Ateneo de Naga University. It includes student voting flows, election results, administration screens, analytics, candidate management, and branded public pages.

## Project Structure

```text
VoteHub/
├── frontend/   # Vite + React application
└── backend/    # Backend service workspace
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```

Build the frontend:

```bash
cd frontend
npm run build
```

## Backend

The backend workspace is prepared in `backend/`. Add API, database, authentication, and deployment code there as the server implementation is introduced.
